# Database Operations Management System

A Node.js and MySQL project that demonstrates core database management functionalities, including database connectivity, table creation, record insertion, retrieval, updating, deletion, and schema modification.

## Technologies Used

* Node.js
* MySQL
* JavaScript

## Features

* Database and table management
* CRUD operations (Create, Read, Update, Delete)
* SQL query execution through Node.js
* Table schema modifications

## Run the Project

```bash
npm install mysql2
node assignment.js
```

Developed to showcase practical integration between Node.js and MySQL for efficient database operations.
